## shootarc-0.1
### Arcade space-shooter written in C and SDL2
  
#### Controls
##### Right-handed
**Steering**: W A S D  
**Turbo**: comma  
**Shoot**: space

##### Left-handed
**Steering**: Arrow keys  
**Turbo**: z  
**Shoot**: space
