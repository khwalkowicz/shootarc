## arcshoot-0.2
### Arcade space-shooter written in C and SDL2
  
#### Controls
##### Right-handed
**Menu movement**: W A S D **or** arrow keys to select, enter or space
to accept  
**Steering**: W A S D **or** arrow keys  
**Shoot**: space  
**Pause**: esc
